# アップとルーズで伝える 学習アプリ

小学4年 国語（光村図書）の説明文「アップとルーズで伝える」（中谷日出）を学習するための React アプリです。読む／問題／漢字／構成マップ／対比表 の5モードで、段落の役割・対比・筆者の主張を段階的に理解できます。

View your app in AI Studio: https://ai.studio/apps/d40a24f2-53ab-4444-a5c7-55974261c9fb

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
